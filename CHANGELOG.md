# Changelog

## 0.5.0

- Renamed the extension to ModelLane.
- Consolidated built-in VS Code Chat models under one `ModelLane` provider.
- Added `ModelLane Refresh` status bar button for one-click local model sensing and refresh.
- Added support for LM Studio, Ollama, vLLM, llama.cpp, and Llamafile model sources.
- Shows only loaded/running callable models in the built-in VS Code Chat picker.
- Kept local model endpoints loopback-only by default.
